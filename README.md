 ### How to run the code

Command line usage for object detection using YOLOv3 

* Python

  * A single image:
    	

    ```bash
    python3 Video_Detection.py --image=city.jpg
    ```

  * A video file:

       ```bash
       python Video_Detection.py --video=run.mp4
       ```
